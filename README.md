## Run Locally

Clone the project

```bash
  git clone https://github.com/DaviDemarquiBraga/kernel-network
```

Go to the project directory

```bash
  cd kernel-network
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
